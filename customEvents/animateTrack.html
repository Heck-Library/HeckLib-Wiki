AnimateTrack

This page contains all the information needed to know about the AnimateTrack event in HeckLib.

Usage

The following example shows how to create a basic pulse scale animation for all objects on track "foo". The animation would occur on beat number 4 in this case.

new AnimateTrack(4, {
    track: "foo",
    duration: 1,
    scale: [
        [2, 2, 2, 0],
        [1, 1, 1, 1]
    ]
}).push();

Parameters

All of the parameters you can use in an AnimateTrack event are in the table below with descriptions. If you wish to learn more about these, they are fully documented in the JSDoc which can be seen in the editor through IntelliSense. (IntelliSense is Visual Studio Code's autocompletition dropdown)

Parameter Type Description
track string | string[] The track to be animated by the event. (This value is the only value that is necessary for the event to be generated.)
duration number The duration of the animation in beats.
attenuation vec1anim The fog attenuation animation. This is only for V2, if you want to control the fog in V3, please use AnimateComponent instead of an AnimateTrack.
height vec1anim The fog height animation of the track. This is only for V2, if you want to control the fog height in V3, please use AnimateComponent instead of an AnimateTrack.
offset vec1anim The fog offset animation of the track. This is only for V2, if you want to control the fog offset in V3, please use AnimateComponent instead of an AnimateTrack.
startY vec1anim The fog startY animation of the track. This is only for V2, if you want to control the fog in V3, please use AnimateComponent instead of an AnimateTrack.
color vec4anim The color animation of the track.
dissolve vec1anim The dissolve animation of the track. Please note that this does not dissolve the arrows of notes, you have to dissolve them separately using dissolveArrow.
dissolveArrow vec1anim The dissolve arrow animation of the track. Since this dissolves only arrows, it works only with notes, nothing else.
localPosition vec3anim The local position offset animation of the track. This only works on environment objects and honestly, I don't know the difference between this and regular position...
position vec3anim The position offset animation of the track. Since V3, this has been only for environment objects as an absolute position. If you wish to animate notes' or walls' position offset in V3, please use offsetPosition instead. Default position is xyz = 0.
offsetPosition vec3anim The position offset animation of the track. This is only for V3 and works only for walls and notes. This replaces the V2's position for map objects. Default position is xyz = 0.
localRotation vec3anim The local rotation offset animation of the track. This rotates the object around its own rotation anchor which is the bottom left corner for walls and the center of the object for notes. Default rotation is xyz = 0.
rotation vec3anim The rotation offset animation of the track. Since V3, this has been only for environment objects as an absolute rotation value. If you wish to animate notes' or walls' position offset in V3, plase use offsetWorldRotation instead. Default rotation is xyz = 0.
offsetWorldRotation vec3anim The rotation offset animation of the track. This is only for V3 and works only for walls and notes. This replaces the V2's rotation for map objects. Default rotation is xyz = 0.
scale vec3anim The scale offset animation of the track. Default size is xyz = 1.
interactable vec1anim The interactable animation of the track. 1 is interactable, 0 is uninteractable, there is no in between or anything outside that range.
time vec1anim The time animation of the track. This is very complex and highly advised to avoid unless you know what you're doing and absolutely need it.
